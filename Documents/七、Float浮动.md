### 1.1 什么是 CSS 浮动

1. 浮动元素只能在水平方向浮动

+ `float: left` or `float-right`;

2. 浮动元素将会脱离文档流；

2. CSS float 属性定义元素在哪个方向浮动，浮动元素会生成一个块级框，直到该块级框的外边缘碰到包含框或者其他的浮动框为止；
